## § B. World Bank WDI macro (REQUIRED)

Five indicators × 114 countries (113 partners + Kosovo XKX) × 2009–2024.

### Option B.1 — let the pipeline fetch it (recommended)

After installing § A, just run `python -m src.run_phase1` from the
`dissertation/` directory. The `fetch_wdi` function uses `wbdata` to pull
directly from the World Bank API and caches the result to
`data/raw/wdi_macro.csv`. No manual download needed if your machine can reach
`api.worldbank.org`.

### Option B.2 — manual bulk CSVs (fallback)

If your network blocks the World Bank API, download the 5 indicator ZIP files
from https://databank.worldbank.org/source/world-development-indicators or the
per-indicator pages:

1. GDP (current US$): https://data.worldbank.org/indicator/NY.GDP.MKTP.CD
2. GDP growth (% annual): https://data.worldbank.org/indicator/NY.GDP.MKTP.KD.ZG
3. Population, total: https://data.worldbank.org/indicator/SP.POP.TOTL
4. FDI, net inflows (% of GDP): https://data.worldbank.org/indicator/BX.KLT.DINV.WD.GD.ZS
5. Inflation, consumer prices (% annual): https://data.worldbank.org/indicator/FP.CPI.TOTL.ZG

On each page click **Download → CSV**. Save all 5 ZIPs into
`data/raw/wdi_bulk/` (create the folder if needed). Then run the helper:

```bash
python -m scripts.consolidate_wdi
```

That script unpacks the 5 ZIPs, pivots them to long format, and writes
`data/raw/wdi_macro.csv` in the exact schema `data_pipeline.fetch_wdi`
produces. (Script provided at `scripts/consolidate_wdi.py`.)

## § C. CEPII GeoDist (REQUIRED)

1. Open http://www.cepii.fr/CEPII/en/bdd_modele/bdd_modele_item.asp?id=6 and
   accept the EULA, or the direct zip:
   http://www.cepii.fr/distance/dist_cepii.zip
2. Unzip. The archive contains `dist_cepii.dta` and `dist_cepii.xls` (and
   sometimes `dist_cepii.csv`). Any of these works; convert to CSV if needed:

   ```python
   import pandas as pd
   # Stata file (preferred — cleanest encoding):
   pd.read_stata("dist_cepii.dta").to_csv("data/raw/cepii_geodist.csv", index=False)
   # OR from the Excel file:
   pd.read_excel("dist_cepii.xls").to_csv("data/raw/cepii_geodist.csv", index=False)
   ```

3. Verify: `wc -l data/raw/cepii_geodist.csv` → should be ~50,200 rows
   (225 × 224 reporter-partner pairs).

Kosovo note: CEPII GeoDist does **not** include Kosovo as a reporter. The
pipeline detects this and falls back to a Haversine distance computation using
`data/raw/capital_coords.csv` (already seeded with the 20+ partners where
Kosovo exchange volumes matter; extend if needed). Contiguity and common
language for Kosovo dyads are hand-curated in `resolve_kosovo_distances`
(Albanian-speaking: AL, MK, ME; Serbian-speaking: XS, ME, BA; land borders:
AL, MK, ME, XS).

## § D. Gap Institute 2019 PDF (OPTIONAL — validation only)

The seed CSV at `data/raw/gap_institute_diversion.csv` already encodes the
three headline figures cited in the plan (cereals 83 %, foodstuffs 62 %,
beverages 55 %). Downloading the PDF lets you reconcile these with the source
and add more sector-level detail if desired.

- Report: "Analysis of the Impact of the 100 % Tax on Goods Imported from
  Serbia and Bosnia and Herzegovina", Gap Institute / Institituti GAP, 2019.
- Search: https://www.institutigap.org/ → Publications → 2019.
- Save as: `data/raw/gap_institute_2019.pdf`

After download, open and re-check the three percentages against the CSV; if
any differ, edit `data/raw/gap_institute_diversion.csv` directly. No code
change required.

## § E. Eurostat COMEXT HS2 (OPTIONAL)

Per Plan §7.1 Task C and `EUROSTAT_DECISION.md`, this dataset is optional.
With it enabled, Dashboard Page 2 gets the bilateral × sector diversion
charts; without it, that section renders hidden and no bilateral × sector
claim is made in Chapter 5.

1. https://ec.europa.eu/eurostat/databrowser/view/DS-045409/default/table
2. In the filter panel set:
   - `PARTNER = Kosovo (XK)`
   - `PRODUCT = Top level (HS2) chapters` (01 … 99)
   - `FLOW = Imports` + `Exports`
   - `TIME_PERIOD = 2010 … 2024`
   - `REPORTER = All EU-27 reporters`
3. Click **Download → CSV** (not XLS). The file is ~20 MB.
4. Save as `data/raw/eurostat_comext_hs2.csv`. The pipeline expects columns
   `partner, reporter, hs2, year, flow, value_eur`; light renaming may be
   needed after download (the Eurostat CSV uses `Partner`, `Reporter`, etc.).

A cleaning helper can be added if this dataset is included later.
